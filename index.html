Алгоритмы управления

А. Следование по линии
  • Движение прямо — центральный датчик на линии → одинаковая скорость всех ведущих колёс
  • Коррекция — линия смещается → уменьшение скорости одной стороны для плавного возврата
  • Поиск линии — датчики потеряли маршрут → медленное вращение до обнаружения
Б. Алгоритм безопасности
  • Постоянная проверка сигнала датчика KY-032 в основном цикле программы
  • При обнаружении объекта → немедленная остановка моторов
  • Ожидание освобождения маршрута → возобновление движения
В. Управление подносом
  • Отдельная функция управления вторым L298N (привод подноса)
  • Команда «поднять» → мотор вращает передачу → платформа поднимается
  • Команда «опустить» → мотор в обратную сторону → платформа опускается
Г. Язык программирования
C++ · Arduino IDE

// RoboWaiter Challenge — Официальные правила

Условия соревнования

Создание и демонстрация автономного робота-официанта, способного без вмешательства человека безопасно и точно доставлять заказы в условиях, приближённых к реальному ресторану (школьная столовая).
01
Участники
Команды из 4–6 учащихся 9-х классов. Опциональный наставник из 10–11 классов.
02
Размеры робота
Макс. 40×40×150 см. Минимальная высота подноса — 40 см. Площадь подноса — не менее 400 см². Нагрузка — до 2 кг.
03
Формат
3 заезда для каждой команды. В каждом заезде — 2 попытки. Оффлайн, школьная столовая.
04
Карантинная зона
После 30 минут настройки роботы сдаются в карантин. Прикасаться к роботу и загружать программы запрещено.
05
Маршрутная линия
Ширина линии — 19 мм. Алаң: старт → приём заказа → доставка (4 зоны) → возврат.
06
Кнопки управления
Робот должен иметь кнопки с номерами зон доставки для приёма заказа и возврата.
// Задание робота — пошагово
1
Старт из стартовой зоны — движение по линии к зоне «приёма заказа»
2
Остановка в зоне приёма — ожидание нажатия кнопки с номером зоны доставки
3
Движение к нужной зоне доставки — безопасная транспортировка заказа (до 2 кг)
4
Остановка в зоне доставки — уведомление посетителей (звук / мигание LED)
5
Посетитель нажимает номер своего столика → робот возвращается в стартовую зону
!
При появлении препятствия на маршруте — немедленная остановка и ожидание освобождения пути
